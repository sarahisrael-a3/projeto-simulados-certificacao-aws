## 🚀 COMEÇAR EM 3 PASSOS

### Passo 1: Clone o Repositório (2 min)

```bash

cd projeto-simulados-certificacao-aws
```

### Passo 2: Instale Dependências (5 min)

```bash
npm install
```

### Passo 3: Rode o Projeto (3 min)

```bash

```

**Pronto!** Seu projeto está rodando localmente. 🎉

## ⚠️ PONTOS IMPORTANTES

### Para Iniciantes

```
✅ FAÇA:
- Leia documentação
- Clone e rode localmente
- Faça perguntas no GitHub Discussions
- Procure por issues com label "good first issue"
- Comece com algo pequeno

❌ NÃO FAÇA:
- Comece pelo código mais complexo
- Pule etapas de documentação
- Tenha medo de perguntar
- Envie PR sem testar localmente
```

### Sobre Questões Novas

```
✅ Questão BOM:
- Tem fonte técnica
- Pergunta é clara
- Resposta é inequívoca
- Sem pegadinhas fracas
- Verificado contra duplicatas

❌ Questão RUIM:
- Sem fonte
- Pergunta ambígua
- Múltiplas respostas "certas"
- Pegadinha gramatical
- Duplicada de outra existente

Use: validate_contribution.py ANTES de enviar!
```
