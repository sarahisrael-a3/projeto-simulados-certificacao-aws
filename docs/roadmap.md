# 🗺 Roadmap do Projeto

Nossa visão é evoluir este simulador até que ele se torne a **Cloud Certification Study Tool** de referência para a comunidade.

### 🟢 Fase 1: MVP Funcional (Status: Concluído/Atual)
- [x] Motor de Quiz básico em JavaScript (`quizEngine.js`).
- [x] Estrutura de dados em JSON.
- [x] Suporte inicial a certificações (Cloud Practitioner, Solutions Architect).
- [x] Scripts Python de automação e tradução.

### 🟡 Fase 2: Engajamento e UX (Próximos Passos)
- [ ] Implementar sistema de Pontuação e Histórico de Desempenho (uso do `storageManager.js`).
- [ ] Adicionar Flashcards de revisão.
- [ ] Melhorar a interface do usuário (UI/UX responsivo).
- [ ] Sistema de categorias e níveis de dificuldade por questão.

### 🔵 Fase 3: Analytics e Gamificação
- [ ] Dashboard de progresso do usuário com gráficos (`chartManager.js`).
- [ ] Sugestão inteligente de estudos baseada nos erros do simulado.
- [ ] Relatório exportável de desempenho.

### 🟣 Fase 4: Escalabilidade e Plataforma
- [ ] Backend para ranqueamento (Leaderboard da guilda).
- [ ] Autenticação de usuários.
- [ ] Abertura da ferramenta para estudantes externos à empresa.
